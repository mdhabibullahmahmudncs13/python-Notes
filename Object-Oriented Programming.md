# Object-Oriented Programming

OOP is a programming paradigm based on objects that contain data and code.

## Four Pillars of OOP

1. **Encapsulation**: Bundle data and methods together
2. **Inheritance**: Create new classes from existing ones
3. **Polymorphism**: Objects of different types through same interface
4. **Abstraction**: Hide complex implementation details
