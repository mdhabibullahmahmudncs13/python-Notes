# Control Flow

Control flow determines the order in which code executes, allowing programs to make decisions and repeat actions.

## Topics

- [[Python If Else]] - Conditional statements
- [[Python Match Statements]] - Pattern matching (Python 3.10+)
- [[Python While Loops]] - Repeat while condition is true
- [[Python For Loops]] - Iterate over sequences

## Control Flow Keywords

- `if`, `elif`, `else` - Conditional execution
- `match`, `case` - Pattern matching
- `for` - Iterate over sequence
- `while` - Loop while condition is true
- `break` - Exit loop
- `continue` - Skip to next iteration
- `pass` - Placeholder (do nothing)
