# Python Get Started

Setting up Python on your computer and writing your first program.

## Installation

### Check if Python is installed
```bash
python --version
# or
python3 --version
```

### Download Python
Visit the official Python website: https://www.python.org/

### Install Python
- Windows: Download installer and run
- Mac: Use installer or Homebrew
- Linux: Usually pre-installed, or use package manager

## Running Python

### Interactive Mode (REPL)
```bash
python
```

### Script Mode
Create a file `hello.py`:
```python
print("Hello, World!")
```

Run it:
```bash
python hello.py
```
