# Python Intro

Python is a popular programming language created by Guido van Rossum and released in 1991.

## What is Python?

Python is:
- An interpreted language
- High-level and general-purpose
- Known for its clean and readable syntax
- Dynamically typed
- Object-oriented

## What can Python be used for?

- Web development (server-side)
- Software development
- Mathematics and scientific computing
- System scripting
- Data analysis and visualization
- Machine learning and AI
- Automation

## Why Python?

- Works on different platforms (Windows, Mac, Linux)
- Simple syntax similar to English
- Allows developers to write programs with fewer lines
- Runs on an interpreter system (immediate execution)
- Can be procedural, object-oriented, or functional

## Python Versions

- Python 2 (legacy, no longer supported)
- Python 3 (current version, recommended)
